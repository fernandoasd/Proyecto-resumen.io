Consola

En Google Chrome la consola se abre con Ctrl + Mayus + I.

Usar console.clear() para limpiar ventana de consola.

Usar console.log("Texto en consola") para mostrar mensaje en consola

Usar console.error("Error") para mostrar mensaje rojo de error en consola.

Usar console.warn("Advertencia") para mostrar mensaje amarilo de advertencia en consola.
