Consola

En Google Chrome la consola se abre con Ctrl + Mayus + I.

Usar console.clear() para limpiar ventana de consola.

Usar console.log("Texto en consola") para mostrar mensaje en consola

Usar console.error("Error") para mostrar mensaje rojo de error en consola.

Usar console.warn("Advertencia") para mostrar mensaje amarilo de advertencia en consola.

Usar console.log("%cMensaje de color verde y grande", "color:yellow; fontsize: xlarge") para mostrar mensaje amarillo tamaño grande? (no funciona el)


Declaración de variables y operaciones:

Usar var nombre_variable para declarar variable (no puede empezar con número y solo ciontener los carácteres _ y $).

Usar let nombre_variable la variable solo afecta a su ámbito, ya que son locales al bloque donde se están declarando.

Usar const nombre_variable "const" no declara una variable sino una constante, que no puede variar su valor a lo largo de la ejecución de un programa.


Ejemplo: var variable1=23, variable2=33 para declarar variables, var sumatotal= sumando1 + sumando2 para declarar la variable de la suma

Impresión del resultado en consola (en color verde): console.log("%cEl resultado de la suma de es " + sumatotal + ".", "color:#0f0")

Lo que se ve en consola (presionar f12):

El resultado de la suma de es 35.

Git + GitHub.

En la carpeta donde se quiere iniciar el repositorio dar doble click a Git Bash Here.

Escribir git version para saber la versión del Git instalada.

Escribir git init para iniciar repositorio local.

Escribir git add index.html para agregar archivo al staging area, en este caso "index.html" .

Escribir git add . para agregar todos los archivos al staging área.

Escribir git comment -m "comentario del commit" para hacer un commit.

Escribir git remote add origin https:// ... .git para agregar repositorio locar al repositorio remoto en HitHub.

Escribir git status -s o git status para ver los archivos modificados despues del ultimo commit.

Escribir git log --oneline para ver todos los commits, notar que cada uno tiene un codigo único.

Escribir git reset --hard 01cdac6 para cargar a un commit anterior, en este caso al 01cdac6.

    Escribir git commit -amend para editar VIM: modificar comentarios de commits.

  • Escribir :i para modificar comentario, luego otra vez :i para guardar.
  • :wq para guadar cambios y salir del VIM.

Escribir git tag v1 -m "comentario del tag: Version 1 del proyecto" paraagregar un tag o version del proyecto.

Escribir git push --tags para subir tags a repositorio remoto.

Escribir git branch nombre_del_branch para crear una rama o branch.

Escribir git branch para consultar todos los branch, el principal es el main por defecto.

Escribir git checkout nombre_del_brancg para moverse al branch nombre_del_branch.

Escribir git merge rama_a_unir para hacer un merge con la rama main. Es necesario estar posicionado en el main para hacerlo.

Escribir git branch -d nombre_branch para borrar branch.
